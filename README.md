This project is an interactive Sales Performance Dashboard built in Microsoft Excel using pivot tables, slicers, charts, and KPI cards. It analyzes sales data across regions, products, customer segments, and time periods to provide clear insights for business decision-making.

🚀 Project Highlights
Interactive slicers for Month, Region, Product, and Gender
Clean and modern dashboard layout
Pivot Charts and Pivot Tables for quick analysis
KPI Cards showing important metrics:
Total Sales
Total Profit
Number of Orders
Profit %
📈 Key Insights
East region records the highest sales
Sofa & Laptop categories contribute major profits
Male customers form the largest share of buyers
Peak sales occur around September
🛠️ Tools & Techniques Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Filters
Data Cleaning
Dashboard Design & Formatting
